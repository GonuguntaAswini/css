Mobile-first: stacked vertically on small screens, becomes 4 in a row on desktop. Try resizing your browser.

Parent properties used:

  • display: flex — creates the flex formatting context.
  • flex-direction — changes main axis (column on mobile, row on desktop).
  • justify-content — aligns children along the main axis.
  • align-items — aligns children along the cross axis.
  • gap — consistent spacing between children (no margin hacks).
